# Getting started

## Create a new venv
> python -m venv venv
> source venv/bin/activate

## Install requiments
> pip install - r requirements.txt

## Run it
> python app\main.py

## Build execute application from .py file
Open command line (CMD) and run:
> cd tools
> py2app.sh
or `-d|--debug` for debug mode with console
> py2app.sh -d
